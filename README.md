### Serial Connector

Приложение-сервис для чтения данных по серийному порту.

Обязательно посетите вики: [Справка и документация](https://github.com/delletenebre/serial_connector/wiki)

## Возможности
Подключение:
  - [x] USB
  - [x] Bluetooth
  - [ ] WebSocket
